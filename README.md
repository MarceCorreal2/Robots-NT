**ROBOTS - NINJA TRADER - INVERBOTS**

Espacio para guardar toda la info del desarrollo del aprendizaje de Bots de trading, iniciando por intentar replicar Inverbot

# Objetivos del Proyecto

•	Entender la lógica detrás de un robot de trading de inverbots (NinjaTrader / NinjaScript).
•	Analizar el comportamiento del robot INVERBOTS como referencia.
•	Documentar paso a paso mi aprendizaje, Documentar toda la información que voy investigando
•	Desarrollar mi propio robot basado en tendencias, entradas técnicas y gestión del riesgo.
•	Crear una base sólida para futuros robots o estrategias automatizadas.
La idea es trabajar con el bot de Inverbots e y bajar toda la informaciòn de ninja trader para tener algo mas de datos.
